# Sentiment Analysis on Book Ratings

## Overview
This project involves sentiment analysis on a dataset of book ratings using the TextBlob library in Python. The goal is to explore the relationship between book ratings and sentiments, providing insights into the emotional tone of user reviews.

## Key Features
- **Sentiment Analysis:** Utilized the TextBlob library to perform sentiment analysis on 'review title' and 'review description' data.
- **Top 10 Positive Books:** Identified and listed the top 10 books with the highest positive ratings.

## Tools and Technologies
- Python
- pandas
- TextBlob
- matplotlib
- seaborn
